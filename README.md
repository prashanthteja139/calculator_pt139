# Python Calculator

A simple Python calculator application that supports basic arithmetic operations and more advanced functionalities. This calculator can handle addition, subtraction, multiplication, division, and exponentiation. It also supports multiple inputs for addition and multiplication operations.

## Features

- **Addition**: Supports multiple numbers.
- **Subtraction**: Handles two numbers.
- **Multiplication**: Supports multiple numbers.
- **Division**: Handles two numbers.
- **Exponentiation**: Calculates base raised to an exponent.

## Installation

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/prashanthteja139/calculator_pt139.git
   ```
  

2. **Navigate to the Project Directory**:
   ```bash
   cd calculator_pt139
   ```

3. **Run the Calculator**:
   ```bash
   python calculator.py
   ```

## Usage

1. **Select Operation**:
   When prompted, select the desired operation by entering the corresponding number:
   - 1 for Addition
   - 2 for Subtraction
   - 3 for Multiplication
   - 4 for Division
   - 5 for Exponentiation

2. **Input Numbers**:
   Follow the prompts to enter the numbers for the selected operation.

3. **View Result**:
   The result will be displayed after each calculation.

4. **Continue or Exit**:
   You can choose to perform another calculation or exit the program.

## Example

To perform an addition of multiple numbers:
1. Choose option `+` for addition.
2. Enter the numbers separated by space, e.g., `5 10 15`.

To perform an exponentiation:
1. Choose option `e` for exponentiation.
2. Enter the base and exponent.
